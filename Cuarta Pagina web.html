javascrip

Los botones se crean en HTML con la etiqueta y su interactividad se añade con JavaScript. HTML define el botón y su contenido, mientras que JavaScript añade la funcionalidad que se ejecuta cuando el usuario hace clic en él, como cambiar el texto, abrir una nueva página o modificar elementos de la página. Para esto, se usan los atributos HTML como onclick para llamar a una función de JavaScript.
